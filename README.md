
Project Title:
FASTag System

Project Overview:
FASTag is an electronic toll collection system operated by the National Highway Authority of India (NHAI). It uses Radio Frequency Identification (RFID) technology to make toll payments directly from a prepaid account linked to the FASTag sticker attached to the vehicle’s windscreen.

This project simulates a FASTag management system that allows users to:
- Register vehicles and assign FASTags
- Recharge FASTag accounts
- Automatically deduct toll amounts
- Generate transaction receipts

Key Features:
1. User Registration and Login
   - Secure login for users and administrators.
   - Vehicle registration and FASTag activation.

2. Balance Management
   - Users can recharge their FASTag accounts.
   - Automatic deduction at toll gates.




Modules:
- User Module – login, recharge.
- Transaction Module – Handles real-time deductions and history logs.

Technologies Used:
- Frontend: HTML, CSS, JavaScript,Tailvent css.
